# RC18.2 Change Report

## Outcome

Rental Car Trip Info now has two unambiguous navigation actions: one for the Christchurch pickup depot and one for the Queenstown return depot. The Rental Car Office and depot cards now wrap long text cleanly on narrow mobile screens.

## Navigation binding

- Pickup: `79 Stanleys Road, Harewood, Christchurch`
- Return: `2/13 Red Oaks Drive, Frankton, Queenstown`
- Removed the generic `Navigate to depot` action.
- Preserved the separate shuttle collection note for `264 Russley Road, Avonhead, Christchurch`.

## Mobile layout

- Rental depot cards switch to one column at widths up to 520 px.
- Depot addresses, office email, and long booking text wrap instead of clipping.
- Navigation buttons occupy the full card width and allow wrapped labels.

- Version: RC18.2

No itinerary decisions, Guide content, Expenses, Moments, Admin Mode, exports, sync logic, manifest authority, or Service Worker strategy were changed.
